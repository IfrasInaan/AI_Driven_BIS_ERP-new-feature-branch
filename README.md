# IfrasInaan-AI_Driven_BIS_ERP_System-FYP-
AI-Driven ERP System is a Final Year Project from PAF-IAST, Pakistan. It integrates ERP, SCM, and CRM modules, automates business workflows, and provides AI-based demand forecasting, inventory management, and analytics for smarter decisions, built with ASP.NET MVC, SQL Server, and Python.
